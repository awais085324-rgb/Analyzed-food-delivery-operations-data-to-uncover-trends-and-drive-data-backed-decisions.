# Analyzed-food-delivery-operations-data-to-uncover-trends-and-drive-data-backed-decisions.
🚀 Project Overview

The dashboard highlights key performance metrics such as:

💰 Total Sales: $1.20M
⭐ Average Rating: 3.92
🛒 Number of Items: 9K+
📦 Average Sales per Item: $141

It provides a clear view of sales trends, outlet performance, and customer preferences.

📈 Key Insights
Tier 3 outlets generate the highest sales contribution
Peak sales observed around 2018
Low-fat items dominate overall sales
Supermarket Type 1 has the highest item count and revenue
🛠️ Tools & Technologies
Power BI
Data Cleaning & Transformation
Data Visualization
📂 Dataset

Raw sample dataset was used and transformed within Power BI using data modeling and DAX measures.

📌 Features
Interactive filters (Outlet Size, Location, Item Type)
Sales trend analysis over time
Category-wise and outlet-wise performance
